무역통계를 활용한 기술개발 유망품목발굴 시스템 개요

목적
국제무역데이터를 기반으로 글로벌공급망(GVC)의 변화를 계량적 지수로 측정하고 이를 활용한 기술개발 유망품목발굴에 필요한 정보 제공
내용
무역거래품목코드(HS code) : 국제무역거래 품목별 상품내용의 검색이 가능
무역통계지수(Index) : 무역거래품목 중 품목별 고부가가치수준과 국가별 수입비중 검색 가능
무역이슈분석(Target) : 특정 목표시장의 공략 전략에 따른 기술개발 품목 추출이 가능
